Veriler her yeni şarkıda otomatik güncellenir.

Verileri sitenize entegre etmek için User ID kullanın.

1. Canlı Durum (Live)

Şu an çalan şarkı bilgileri:

GET /api/status/[USER_ID]
  • title / artist: Şarkı ve Sanatçı adı.
  • artwork: Kapak resmi URL'si.
  • isPlaying: Çalma durumu (true/false).
  • currentTime / totalTime: "1:23" formatında süre bilgisi.
  • totalTimeSeconds: Toplam süre (Saniye).
  • totalListens: Toplam dinlenme sayısı.
2. İstatistikler (Stats)

Kullanıcının genel verileri:

GET /api/stats/[USER_ID]
  • history: Son dinlenen 10 şarkı listesi.
  • topSongs: En çok dinlenen 5 şarkı.
  • topArtists: En sevilen 3 sanatçı.
  • monthlyCount: Bu ayki toplam dinleme.
Entegrasyon Örneği (JS)
fetch(`${API}/status/${USER_ID}`) .then(r => r.json()) .then(data => { if(data.isPlaying) { console.log("Dinliyor: " + data.title); } });

* API Key gizlidir, sadece eklenti içindir.
* User ID açıktır, sitenizde kullanabilirsiniz.

  • Dinlediğiniz her şarkı otomatik olarak veritabanına kaydedilir.
  • Şarkı adı ve sanatçı algılandığında yeni kayıt oluşturulur.
  • En çok dinlediğiniz sanatçı ve şarkıları görebilirsiniz.
  • Web sitenizde "Şu an dinliyorum" widget'ı yapabilirsiniz.
